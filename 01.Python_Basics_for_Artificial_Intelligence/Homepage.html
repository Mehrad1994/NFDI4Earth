About This Course

The course “Python: Basics for Artificial Intelligence” introduces participants to fundamental Python programming skills within an AI context. Learners will set up their development environment (Anaconda, VS Code), manipulate and analyze data using NumPy and Pandas, and implement basic programming logic to solve AI-related problems. (Access this course on GitLab).

Level

Introductory / Intermediate

Requirements

Moderately powerful computer

Subject Area

Artificial Intelligence, Geomatics, Geographic Information Science, Machine Learning

What You Will Learn

  • Getting started with Anaconda & VS Code
  • Using Jupyter Notebooks in VS Code
  • Python variables
  • Lists: basics, indexing, slicing
  • For-loops
  • Conditional statements (if/elif/else)
  • Combining loops & conditions
  • Functions in Python
  • Combining functions, loops & conditions
  • NumPy basics: arrays & operations
  • Pandas basics: Series, DataFrame, data selection
  • Advanced Pandas: conditional selections, grouping, import/export

Resources

Artificial intelligence – basics and geographical applications by Torben Dedring, Lars Tum, Andreas Rienow (Institute of Geography, Working Group of Interdisciplinary Geographic Information Sciences, 2024; CC BY-NC-SA 4.0).
