About This Course

The course “Python: Basics for Artificial Intelligence” introduces participants to the fundamentals of Python programming within the context of AI development. Participants will learn to set up their development environment, manipulate and analyze data using libraries like NumPy and Pandas, and solve real-world problems. Through practical exercises and examples, participants will build a solid foundation for AI workflows and gain hands-on experience in data handling, algorithmic thinking, and basic model construction (Access this course on GitHub).

Level

Introductory, Intermediate

Requirements

Moderately powerful computer

Subject Area

Python, Data Science Libraries

What You Will Learn

  • Getting Started – set up your Python environment with Anaconda VS Code and Jupyter
  • Python Basics – master core programming concepts such as variables lists loops conditions and functions
  • Python NumPy – manipulate numerical arrays and perform efficient vectorized operations
  • Python Pandas – analyze and transform tabular data using DataFrames
