About This Course

The course “Python: Basics for Artificial Intelligence” introduces participants to the fundamentals of Python programming within the context of AI development. Participants will learn to set up their development environment, manipulate and analyze data using libraries like NumPy and Pandas, and implement programming logic to solve real-world AI problems. Through practical exercises and examples, participants will build a solid foundation for AI workflows and gain hands-on experience in data handling, algorithmic thinking, and basic model construction. (Access this course on GitLab).

Level

Introductory, Intermediate

Requirements

Moderately powerful computer

Course Curriculum

Lesson 1: Getting Started with Anaconda and Visual Studio Code

Lesson 2: Getting Started with Jupyter Notebooks in VS Code

Lesson 3: Introduction to Variables in Python, Exercise

Lesson 4: Understanding Lists in Python: Basics, Indexing, and Slicing, Exercise

Lesson 5: Introduction to For-Loops in Python: Iteration and List Processing, Exercise

Lesson 6: Understanding Conditional Statements in Python: If, Elif, and Else, Exercise

Lesson 7: Combining Loops and Conditions in Python, Exercise

Lesson 8: Introduction to Functions in Python, Exercise

Lesson 9: Combining Functions, Loops, and Conditions, Exercise

Lesson 10: Introduction to NumPy: Arrays, Operations, and Data Manipulation, Exercise

Lesson 11: Introduction to Pandas: Series, DataFrames, and Data Selection, Exercise

Lesson 12: Advanced Pandas: Conditional Selections, Grouping, and Data Import/Export, Exercise
