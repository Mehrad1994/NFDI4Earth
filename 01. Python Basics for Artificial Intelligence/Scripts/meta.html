About This Course

This comprehensive course offers an introduction to Python programming and its applications in Earth sciences. Combining theoretical foundations with hands-on exercises and practical applications in Jupyter notebooks, it guides learners through variables, data structures, control flows, and core AI concepts. Participants will explore machine learning and deep learning models, from Random Forest to CNNs and LSTMs. (Access this course on GitLab).

Level

Introductory

Requirements

Moderately powerful computer, internet access, optional GIS familiarity

Subject Area

Python Programming, GIS, Machine Learning, Deep Learning, Earth Sciences

What You Will Learn

  • Setting up Python and Jupyter environments
  • Programming basics: variables, data types, loops, and conditions
  • Core AI and machine learning concepts: Random Forest, Neural Networks, CNNs, RNNs
  • Geographical applications: water level prediction, ship detection, soil sealing
  • Hands-on coding exercises and solutions in Jupyter notebooks
